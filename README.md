# Professional Drones - Landing Page

Moderní webová prezentace pro profesionální drony s důrazem na čistý design a plynulé animace.

## 📋 Přehled projektu
Tento web slouží jako jednostránková (one-page) prezentace moderních bezpilotních letounů. Obsahuje sekce pro představení funkcí, technické parametry a ukázky letových režimů.

## 🛠️ Technologie
- **HTML5** - Sémantická struktura obsahu.
- **Tailwind CSS** - Moderní stylování přímo v HTML pomocí utility classes.
- **GSAP (GreenSock)** - Knihovna pro pokročilé animace (využita pro scroll-efekty).
- **Google Fonts** - Typografie (použit výrazný font Anton).

## ✨ Klíčové vlastnosti
- **Hero Sekce:** Dynamický vizuál s vrstveným pozadím a plovoucím modelem dronu.
- **Responzivní Navigace:** Plně funkční menu přizpůsobené pro mobilní telefony i desktop.
- **Statistiky:** Animované počítadlo výkonnostních parametrů (výdrž baterie, dosah).
- **Horizontal Scroll Section:** Interaktivní slider s popisem letových módů (Follow, Orbit, Hyperlapse).

## 📁 Struktura souborů
- `index.html` - Hlavní soubor se strukturou a Tailwind konfigurací.
- `script.js` - Skript pro animace (GSAP & ScrollTrigger).
- `style.css` - Style sheet dodatecny pro tailwind.
- `public/` - Složka obsahující obrázky (drony, pozadí).
